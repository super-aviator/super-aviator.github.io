SpringBoot自定义数据源

发表于 2021-08-16 更新于 2021-10-12 分类于 Spring

在开发中，经常会有用到使用多数据源的情况，SpringBoot框架会自动定义一个配置前缀为spring.datasource开头的默认数据源，如果需要使用到其他非默认的数据源，则需要手动去定义，如何在SpringBoot项目中去自定义数据源是一个开发最基本的开发技能，以前总是会按照网上的博客去配置，现在去系统学习一下吧，后面能够用得上。

使用Mybatis框架时，自定义一个数据源基本包括以下几个步骤：

  1. 配置文件中声明数据库配置
  2. 根据配置文件声明DataSource
  3. 定义DataSourceTransactionManager或者JpaTransactionManager
  4. 定义SqlSessionFactory和SqlSessionTemplate或者定义EntityManagerFactoryBean
  5. 使用@MapperScan或者@EnableJpaRepositories注解区分数据源

Lambda表达式和匿名内部类中无法更改外部值的原因

发表于 2021-04-05 更新于 2021-07-22 分类于 Java

什么情况下无法编译

// 在匿名内部类外更改匿名内部类中使用到的值的时候
int a = 12;
Function<Integer, Integer> f = x -> x + a;
a = 14;

// 在匿名内部类中更改外部的值
int a = 16;
Function<Integer, Integer> f = x -> { a++; return x + a; }

以上两种情况都无法通过编译，匿名内部类中使用到的外部值是无法更改的。
